CSS Typography

Definition

CSS typography is the term to describe how text is styled and shown on a website using CSS. These methods cover all the CSS properties and techniques used to control how text is displayed, making it readable and attractive across the webpage.

Font Examples

  • Font-family - set the font type.
  • Font-size - set the font size.
  • Font-weight - adjust boldness of text from 100-1000. (default value: 400-500)
  • Font-style - commonly used to make text italic. Other values include: normal, oblique.

Text Layout

  • Line-height - padding between lines of text.
  • Letter-spacing - adjust space between single characters (px, rem, etc).
  • Word-spacing - adjust spacing between words.
  • Text-align - align text to left, right, center or justify.

Text Effects

  • Text-decoration - used to add underline, strikethrough (line-through) or overline. Default value: None.
  • Text-transform - used to apply uppercase, lowercase or capitalised cases.
  • Text-shadow - adds shadow to text.
  • Text-indent - adjust indentation (space between margin and first line of text).
  • Color - set text colour.

Advanced Properties

  • Font-variant - small caps and other typographic features.
  • White-space - controls how spaces and line breaks are handled.
  • Direction - set text direction (e.g. ltr, rtl).
  • @font-face - allow custom web fonts to be loaded.
