CSS Typography & Fonts

Definition

CSS typography is the term to describe how text is styled and shown on a website using CSS. These methods cover all the CSS properties and techniques used to control how text is displayed, making it readable and attractive across the webpage.

Internal Fonts

Internal fonts are members of the font family that are pre-installed on computers. Because they default fonts on most IT systems, they can be declared at any time with the 'font-family' CSS property. Common font types used on applications and websites include: Arial, Calibri, Times New Roman, etc.

External Fonts

External fonts are custom fonts from elsewhere (i.e. hosted on servers). They are not pre-installed on most computers; as such, to use an external font, the font file needs to be imported from the source (e.g. Google fonts). Once the font file has been imported, it can now be declared using the 'font-family' CSS property.

External Font Sources

The below are open sources where custom fonts can be imported and used on a website:

  • Google Fonts: http://fonts.google.com
  • Adobe Fonts: http://fonts.adobe.com
  • Font Squirrel: http://fontsquirrel.com
  • Font Awesome (for icons): http://fontawesome.com

CSS Properties

Font Properties:

  • Font-family: [font] - set the font type (used for both internal or external).
  • Font-size: [length] - set the size of font in.
  • Font-weight: [weight] - adjust boldness of text from 100-1000 (default value: 500).
  • Font-style: [style] - used for making text italic, oblique or 'none' if text was previously italic.

Layout Properties:

  • Line-height: [length] - add padding in-between lines of text.
  • Letter-spacing: [length] - adjust space between single characters.
  • Word-spacing: [length] - adjust spacing between words.
  • Text-align: [value] - aligns the block of text to: left, right, center or justify.

Text Effects:

  • Text-decoration: [value] - for applying any of the following: underline, line-through, overline, or resetting to 'none' if text is already underlined (e.g. hyperlinks).
  • Text-transform: [value] - for making all text uppercase, lowercase or capitlised. Capitalised will make the first letter of every word begin with capital letters.
  • Text-shadow: [h-offset] [v-offset] [b-length] [colour] - adds text shadow. H-offset = horizontal length. V-offset = vertical length. B-length = (optional) blur radius, default value is 0. Colour = shadow colour.
  • Text-indent: [length] - adjusts the indentation of the first line of text.
