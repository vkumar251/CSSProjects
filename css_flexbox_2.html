CSS Flexbox

Overview

Flexbox is a CSS layout model that allows us to create flexible, responsive layouts with the help of a single container element and its child elements. A flexbox is first created by having the parent element be declared as the flex container. After that, flex items are nested inside the flex container. Flex containers are commonly applied to div elements.

Parent Properties

Below are properties applied only to the parent flex element - i.e. the flex container:

  • flex-direction: [row, column, row-reverse, column-reverse] - specify the direction to display flex items. Default value: row.
  • flex-wrap: [wrap, nowrap, wrap-reverse] - specify whether to wrap flex items within the container. Commonly used when there's not enough space to fit it inside the container.
  • flex-flow: [flex-direction] [flex-wrap] - flex-wrap and flex-direction both combined into one property. E.g. flex-wrap: row nowrap;
  • justify-content: [flex-start, flex-end, center, space-around, space-between, space-evenly] - align flex items horizontally.
  • align-items: [flex-start, flex-end, center, stretch, baseline, normal] - align flex items vertically.
  • align-content: [flex-start, flex-end, center, stretch, space-around, space-between, space-evenly] - align flex lines instead of flex items.
